# CLAUDE.md — AI Assistant Rules for gws-skill

## Attribution
- **No attribution** in commits, PRs, code comments, or documentation
- No "Generated by", "Co-authored-by", "Built with AI", or similar
- No AI tool credits anywhere in the project

## Code Standards
- Python 3.10+
- Type hints on all function signatures
- Docstrings on public functions
- No `# type: ignore` — fix the types
- JSON output from all scripts (machine-readable for Jarvis)
- No secrets in code — credentials loaded from env/filesystem at runtime

## Commits
- Conventional commits: `feat:`, `fix:`, `docs:`, `refactor:`
- Keep messages concise and descriptive
- No co-author trailers
